Wumpus World

27 Apr, 2016 · Read in about 2 min · (223 Words)

A SE 329 group project. A graphics-based game developed using Java/OpenGL to demonstrate the Wumpus World game, a popular game to study in AI courses. A description of the game can be found here.

View source

The first phase of the project consisted of developing the graphical interface and connecting it with the game logic so a user could play a single round of the Wumpus game. I worked on implementing the OpenGL graphics code and connecting the user’s interactions with the game logic.

The second phase of the project consisted of a networking component to play against friends, as well as AI components so users could challenge an AI player. When playing the game over a network, users could challenge friends to see who could complete the game with the fewest number of moves (or complete it at all). The AI modes allowed users to challenge an AI oppenent in 3 different difficuly levels. I personally worked on the AI component for this phase. The easy difficulty AI essentially moved in breadth-first manner, ignoring the statistical dangers of various squares. The medium difficulty always chose the safest path, but backtracked so it would take more moves. The hard difficulty additionaly chose the safest path, but moved in a depth-first manner so there was little backtracking and it could reach the gold quickly.
