WikiClicki

22 Mar, 2016 · Read in about 1 min · (163 Words)

A SE329 (project management) group project in which we developed a web-based game made in JavaScript and HTML, utilizing D3 graphing utilities.

The wiki-game is where a person starts at a random page on Wikipedia, and tries to get to another game in a certain amount of clicks. My team developed a web application to play this game in a more streamlined fashion. Users could choose a destination page or have one chosen at random. After clicking start, the app chose a random page for them to start on. Using D3, we showed their current page as a center node, with links to outer nodes for all links on the wikipedia page. In the sidebar, information on the page is shown so users can learn as they explore the depths of wikipedia. Upon clicking an outer node, the center node would be replaced, the current page information would be replaced, and the links on the new page were shown graphically as outer nodes.
