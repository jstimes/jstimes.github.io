Pencil Motion

3 Oct, 2015 · Read in about 1 min · (112 Words)

A C++/OpenGL graphical application that calculates and renders the motion of a pencil tumbling through the air, and bouncing off a surface. This project was for Com S 477 – Applied Problem Solving Techniques for Computer Science.

The class project was just to determine the position and velocity of a pencil tumbling through the air given the pre and post impact velocities and orientations when it hits a surface, with an emphasis on using quaternions to model its rotation. I used OpenGL to render its motion as it tumbles through the air and hits the surface.

This is again a class-wide project, so I can not host the code on my github.
