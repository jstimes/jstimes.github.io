MyReminder

20 Nov, 2014 · Read in about 1 min · (57 Words)

MyReminder was my first ever Android app (other than random experiments). It allowed the user to create reminders by giving a title, and desired time to go off. At that specified time, the app sends the user a notification. Nothing too spectacular, but it was something I was able to use regularly to remind myself of things.
